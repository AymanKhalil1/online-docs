Equations

Frequently codes of practice impose design limits in the form of equations combining more than one stress or displacement type which need to be satisfied at specific locations.
Now ELS version 9’s filtration equations feature is here to help structural designers with this task.

To see how this feature works, follow the following procedures.

Step1: From the filtration browser choose elements equations,
Step2: Add some variables from the add button and they will automatically be named as a letter.
           (The type of this added item is chosen from a drop down.)
Step3: Enter the left hand-side of the equation as if we’re using a hand calculator using the added items.
           Whenever you need to insert one of the defined items just click the add button.
Step4: Choose the sign (<,> or =) from the drop-down & enter the right -hand side value.
Step5: Click apply, Ok.
Step6: Run the animation and watch the elements’ colors change when they satisfy the filtration condition.
           (This feature is not only for elements, it can be used for springs, straining actions and contacts.)

In addition to the filtration equations feature, you can now apply different filtration equations to different views.

Step1:From Window > Choose New Window and tile the windows vertically.
Step2:Select each view and show only one filtration type (elements or springs...etc.).
Step3:Each filter works separately on each window.
