ELS's Viewer's Workspace

ELS's Viewer offers a comprehensive environment to view the structural model with the analysis results. It's your gateway to see the analysis results in details in the form of straining actions, progressive deformation till collapse (if applicable) along with the eigen shapes of the modelled structure. In addition to that, you can view the results as responive charts or contours. The workspace consists of five main sections:

  • Menus
  • Toolbars
  • Browser Panel
  • Control Bar
  • Viewer Window

Menus

Open

The Open Command opens an already existing ELS project.

Close

The Close Command closes the active window.

Save Output

The Save Output Command saves the changes to the active project.

Exit

The Exit Command closes ELS.

Import

Import Drafts

The Import Drafts Command imports drafts from the GUI without the need to run the analysis again.

Import Settings

The Import Settings Command imports a .stng file will previously saved viewer's settings.

Export

DXF Format

It is an exchange format for AutoCAD Drawing files contents. This format enables users to export drawings from ELS Viewer to a dxf format that can be accessed through AutoCAD to ease up the modelling process.

Obj Format

It is a simple data format produced from & imported to ELS and several other programs. It represents the objects' 3D geometry only.

Image

The Image command generates images of the frames of the model in a .png format.

AVI

The AVI command generates a video of the resulting frames of the model in a .AVI format.

Elements Geometry

The Export Elements Geometry Command generates a txt file of the elements found in a specific frame.

STL

The STL (Standard Triangle/Tessellation Language) Option generates a more simplified module of the current ELS model. Exporting a model in an STL format transforms the 3D mesh into a surface mesh composed of triangles only.

Settings

The Export Settings options generates a .stng file containing all the viewer's settings

Script From Open Charts

The Script From Open Charts Option generates a script to automatically create the same chart(s) as the one(s) currently created by importing this script and running it via Run Script command.

All Opened Charts

The All Opened Charts Option creates a txt file of the currently opened charts as a series of coordinates. These can be exported to other programs for analysis or imported into ELS via the Add Static Chart option available in any chart's right click menu.

Visibility Commands

Global Axes

The Global Axes Command shows/hides the three basic axes (X,Y&Z) in the results viewer window.

Pan

The Pan Command enables you to move your view (reposition the view) of the model in the viewer according to your preference without actually moving the model itself.

Global Axes

The Browser Command shows/hides the browser windows on the right side of the viewer.

Zoom Commands

Zoom in

The Zoom in Command increases the magnification(apparent size) of all the objects in the viewer window in one step.

Zoom out

The Zoom out Command decreases the magnification(apparent size) of all the objects in the viewer window in one step.

Zoom Dynamic

The Zoom Dynamic Command zooms in when you drag the model upwards and zooms out when you drag the model downwards.

Zoom Extends

The Zoom Extends Command Adjusts the magnification of the drawing to accommodate the extents of all visible objects.

Zoom Selected

The Zoom Selected Command increases the magnification of the selected object. (Largest possible display of the selected objects)

Zoom Window

The Zoom Window Command increases the magnification of the area specified by the 2 opposite corners of the rectangular selection window.

Zoom Previous

The Zoom Previous Command restores the previous magnified view.

Tip: To quickly zoom in/out press Alt+/Alt- or use the mouse scroll wheel up to zoom in or down to zoom out.

Model Viewing Commands

3D View Isometric

The Isometric Command Displays the full 3D view of the model (The isometric view).

Front (+ve Y-axis)

The Front (+ve Y-axis) Command Displays the front view of the model.

Back (-ve Y-axis)

The Back (-ve Y-axis)Command Displays the back view of the model.

Left (+ve X-axis)

The Left (+ve X-axis) Command Displays the left side view of the model.

Right (-ve X-axis)

The Right (-ve X-axis) Command Displays the front side view of the model.

Bottom (+ve Z-axis)

The Bottom (+ve Z-axis) Command Displays the bottom planar view of the model. (Looking up view)

Top (-ve Z-axis)

The Top (-ve Z-axis) Command Displays the top plan view of the model. (Looking down view).

Rotate Around X

The Rotate Around X Command Rotates the model around X-axis.

Rotate Around Y

The Rotate Around Y Command Rotates the model around Y-axis.

Rotate Around Z

The Rotate Around Z Command Rotates the model around Z-axis.

Rotate Around Screen X

The Rotate Around Screen X Command Rotates the model around the screen's X-axis.

Rotate Around Screen Y

The Rotate Around Screen Y Command Rotates the model around the screen's Y-axis.

Rotate Around Screen Z

The Rotate Around Screen Z Command Rotates the model around the screen's Z-axis.

Rotate Around Screen XYZ

The Rotate Around Screen XYZ Command Gives full ability to fully rotate in all directions.

Arc Ball Rotate

The Arc Ball Rotate Command Gives full ability to fully rotate in all directions around the center of the screen.

Layout Commands

Save Layout

The Save Layout Command enables the user to save his/her preferred layout to be loaded afterwards.

Load Layout

The Load Layout Command loads an already saved layout.

Show/Hide Commands

Show Frame

The Show Frame command shows/hides the external frame.

Elements

The Elements commands control what to be displayed in elements.

Visible Elements

The Visible Elements Command Specifies what elements will be visible: All Elements or Outer Elements or just the Outer Surfaces.

Show Elements

The Show Elements Command Opens up the Show/Hide Elements Settings Dialog box which Specifies the displayed elements based on the chosen material/level.

The Wireframe Mode Activates/Deactivates the wireframe mode.

Shrink Elements

The Shrink Elements Mode Shrinks/Expands the elements so that a virtual gap is made been each two consecutive elements.

Show Elements Number

The Show Elements Number Displays/Hides the elements ID's on the viewer window.

Show Elements Centroid

The Show Elements Centroid Displays/Hides the elements centroids on the viewer window.

Show Boundaries

The Show Boundaries Command shows/hides the boundaries.

Show Springs

The Show Springs Command shows/Hides springs.

Show Link Members

The Show Link Members Command shows/hides the link members.

Show State

The Show State Command shows/hides the various states including the stress state, failure type,cracks,....etc. Note that you should activate "Read Springs States" in the Preferences window.

Show Straining Actions

The Show Straining Actions Command shows/hides the values of straining actions on the model.

Show Sections

The Show Sections Command shows/hides the locations of the sections at which the straining actions are calculated.

Show Contacts

The Show Contacts Command displays small crosses when contact occurs between two objects which means that contact springs are created.

Show Velocity Vectors

The Show Velocity Vectors Command displays the velocity vectors.

Show Blast-Related Data

The Show Blast-Related Data Command displays the blast data including the blast position, the explosive wave propagation,...,etc.

Show Blast Surfaces

The Show Blast Surfaces Command displays the surfaces affected by the blast.

Update Commands

Auto Update Properties

The Auto Update Properties Command automatically updates the results received from the solver.

Update Data

The Update Data Command updates the results received from the solver to get all the results reached till the current moment.

Update Data Settings

The Update Data Settings Command specifies what to be automatically updated once the results are extracted from the viewer.

Selection Commands

Elements

The Elements Command is a tool to only select elements in the model. If not activated, no elements selection can occur.

Springs

The Springs Command is a tool to only select springs in the model. If not activated, no springs selection can occur.

Sections

The Show Springs Commandis a tool to only select sections in the model. If not activated, no sections selection can occur.

Markers

The Markers Command is a tool to only select markers inserted in the model. If not activated, no markers selection can occur.

Show Selected Objects Only

The Show Selected Objects Only Command is a tool to only show the selected objects and hide the remaining parts of the model.

Hide Selected Objects Only

The Hide Selected Objects Only Command is a tool to only hide the selected objects and show the remaining parts of the model.

Show All Objects

The Show All Objects Command is a tool to show the whole model at once.

Hide All Objects

The Hide All Objects Command is a tool to hide the whole model at once.

Selection Range

The Selection Range Command is a tool to select a group of elements or springs using their ID's.

Select by Name

The Select by Name Command is a tool to select an object by name.

Select Elements by Coordinates

The Select Elements by Coordinates Command is a tool to select elements by entering their coordinates.

New Chart Window

The New Chart Window Command creates a chart selectively displaying part of the results in a graphical form.

Show Legend

The Show Legend Command displays the legend dialog box that enables the user to display/hide series based on the selected elements.

Animate

The Animate Command draws the chart step by step in an animated manner connected to the time frame displayed.

Switch X-Y Axes

The Switch X-Y Axes Command rotates the chart such that the data displayed on the horizontal direction are to be displayed on the vertical direction and vice-versa.

Frequency Domain/ Time Domain

The Frequency Domain/ Time Domain Command enables the suer to switch between the time domain and the frequency domain.

Edit Chart Data

The Edit Chart Data Command retrieves the user to the New Chart Dialog to edit the entered data.

Export Chart Data

The Export Chart Data Command exports the data the currently open chart presents.

Chart Options

The Chart Options tool enables the user to customize the way the chart is presented as per his/her preference.

Elements Stress/Strain Settings

The Elements Stress/Strain Settings Command enables the user to present the resulting stresses/strain in a specific direction as eye-friendly colored contours.

Element Kinematics Settings

The Element Kinematics Settings Command enables the user to present the resulting kinematics in a specific direction as eye-friendly colored contours.

Blast Pressure Settings

The Blast Pressure Settings dialogenables the user to present the resulting blast pressure as eye-friendly colored contours.

OpenFOAM Pressure Settings

The OpenFOAM Pressure Settings dialog enables the user to present the resulting OpenFOAM pressure as eye-friendly colored contours.

Show/Hide Commands

Show/Hide Contours

The Show/Hide Contours Command shows/hides the created contours.

Show/Hide Contours Legend

The Show/Hide Contours Legend Command shows/hides the legend of the contours.

Show/Hide Contours Values

The Show/Hide Contours Values Command shows/hides the values of the contours.

View Custom Component

The View Custom Component Command enables the user to define new custom components without which straining actions can't be calculated in components other than rectangular columns and girders. It also allows the user to view all created custom components and their properties and delete existing custom components.

New Eigen Window

The New Eigen Window Command displays the eigen properties(Angular Frequency, Period & Mass Participation Factor) of the structure for each mode along with displaying the animated mode shapes themselves. This command is only available if the user had specified a number of eigen modes in the preprocessing phase in the Output Sets.

New Separation Groups View

The New Separation Groups View commands a new viewing window to display the specified separation groups.

Units

The Blast Pressure Settings dialogenables the user to present the resulting blast pressure as eye-friendly colored contours.

The Statistics dialog displays all the analysis output-related info & statistics.
