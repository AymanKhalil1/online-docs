    ELS's Viewer

    Extreme LoadingÂ® for Structures's (ELS) Viewer is part of ELS's software package, which consists of 3 programs:
    the Modeling Environment ( ELS's Modeler ), ELS's Solver - utilizing the Applied Element Method (AEM), and the Results Viewer ( ELS's Viewer ).

    Program1
    ELS's Modeler
    • Define material types and properties.
    • Define structural component details.
    • Model structure geometry.
    • Apply loads to model.
    Go to ELS's Modeler
    Program2
    ELS's Solver
    • Generates elements and springs for problem.
    • Analyzes the problem using AEM.
    • Writes output files.
    Go to ELS's Solver
    Program3
    ELS's Viewer
    • View and animate structure displacements.
    • Analyze results using charts, contours, and Straining actions diagrams.
    • Export chart data, bitmaps, videos, and many other output formats.
    Go to ELS's Viewer

    Why ELS's Viewer?

    ELS's Viewer is a state-of-the-art graphical user interface that enables you to view structural models with realistic appearance and behavior. It also provides you with powerful tools for analyzing simulation results and presenting them afterwards.



    ELS's Viewer Most Notable Features

    Model Viewing

    • 3D views from any direction.
    • Walkthrough mode.
    • Model clipping for viewing the hidden parts.
    • Any model object can be displayed or hidden.
    • Model elements can be viewed in solid, wireframe, or shrink modes.
    • View spring states.

    Charts

    • Plot data for any data variable against any other variable (including time and loading increments).
    • Manage chart and series display options.
    • Export chart data.
    • Differentiate and integrate any chart data.

    Contour Diagrams

    • Stress/Strain contours can have their values calculated in global and principal axes directions or custom vectors. You can plot them on structure surfaces or crosssections.
    • Kinematic property contours can be drawn for displacement, velocity, acceleration, or rotation.
    • Blast pressure contours on structure surfaces.
    • OpenFOAM pressure on Structure objects.

    Straining Actions Diagrams

    • Draw normal, shear, bending moment, section rotation and torsion diagrams at the same time for different structure parts.
    • Draw Straining actions envelopes.
    • Straining actions values are calculated automatically for sections along columns and girders having automatic mesh.
    • Calculation of straining actions for custom components.

    Eigen Analysis- you can view vibration analysis results

    • View mode shapes and frequencies.
    • Animate mode shapes.
    • Compare mode shapes visually.

    Animation-you can view variations in the following simulation results with respect to time and loading increments

    • Structural displacements.
    • Straining actions diagrams.
    • Eigen modes.
    • Charts.
    • Contours.

    Output Formats - you can export output in many file formats: bitmaps, videos, DXF, STL, Script from opened charts and OBJ format.

    Before You Start

    ELS Viewer is installed as a part of ELS's package and does not have a separate installation procedure. For more information on system requirements and installation of the ELS package see the ELS Quick Start Guide.

    You can access ELS's Viewer through two ways;

    Finding Your Way Through ELS's Viewer

    Before going any further through this manual, you must first understand some ELS and AEM related terms like springs, steel elements, and output frames.




    Springs

    Springs - are the objects generated by ELS to be used in the AEM analysis. The name comes from their behavior which is, more or less, like actual springs in the AEM theory in addition to holding the elements together. There are two ways to define springs:

    1. According to their material (concrete, steel, brick..,etc.)
    2. According to their type; matrix springs (concrete, mortar, brick,â¦â¦) or reinforcement springs or implicit springs.

    All straining actions, stresses, or strains' calculations use these spring objects. Note that link members are also treated as springs having only normal force.
    For more information on springs check out ELS's Theoretical Manual

    Elements

    Elements - are generated by ELS's Solver to be used in the AEM analysis. Structural components are converted to elements connected by springs in the AEM analysis. All components are converted to small elements called *Matrix Elements* except some types of steel sections which are converted to Steel Elements. The conversion depends on the way the steel was modeled in ELS's Modeler, as implicit or explicit steel.
    For more information on element generation and implicit and explicit steel, refer to ELS's Theoretical Manual

    Frames

    Frames - are units which hold the output analysis results. Each frame includes details about the structure at a particular time instance or load increment. Charts, contours, and straining actions can all be displayed at the different frames. Playing the results' simulation shows the structure's response to loading in consecutive frames. Frames are numbered sequentially starting from 1.
